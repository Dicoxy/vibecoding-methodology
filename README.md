## 📁 Шаблоны (templates/)

Копируй в новый проект:

```
templates/
├── CLAUDE.md              ← Главный файл контекста
└── .brief/
    ├── client-brief.md    ← Бриф клиента
    ├── content.md         ← Тексты для сайта
    └── tasks.md           ← Задачи
```

## 📋 Принципы

1. **Улучшаем — не переписываем** (по умолчанию)
2. **Коммит после каждого блока** (чекпоинты)
3. **Escape сразу** (если AI пошёл не туда)
4. **Мокапы перед кодом** (HTML → React)
