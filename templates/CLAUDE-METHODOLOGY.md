## Тир проекта

<!-- Выбери один, удали остальные -->

**Tier 1** (1-5 блоков, один человек)
- Контракты: минимальные (вход/выход/не делает)
- Changelog: один файл, формат по методологии
- RAG: не нужен
- Git: можно без веток

**Tier 2** (8-15 блоков, 2-3 человека)
- Контракты: полные (вход/выход/транспорт/SLA/микросимуляции)
- Changelog: один файл, формат по методологии
- RAG: обязателен
- Git: ветки block/*

**Tier 3** (40+ блоков, команда)
- Контракты: полные + JSON Schema + автотесты
- Changelog: распределённый (per-block + общий)
- RAG: обязателен, распределённый
- Git: ветки block/*, CI/CD

## Правила

### Код
- Одна задача за раз → коммит → СТОП → подтверждение
- Улучшаем существующее, НЕ переписываем
- Агент реализует КОНТРАКТ. Если контракт плохой — эскалация, не самодеятельность.

### Changelog (обязательно)

Формат: ПЛАН (до работы) → ФАКТ (после).

```markdown
## [block-name] | [YYYY-MM-DD] | [агент] | ПЛАН
### Что планируем сделать
- [конкретно]
### Контракт
- Вход: contracts/[name].md v[N]

## [block-name] | [YYYY-MM-DD] | [агент] | ФАКТ
### Сделано
- [конкретно]
### Тесты
- [вход → выход]: ✅/❌
### Отклонения от плана
- [если есть]
```

**Правило:** решение принято → записать СРАЗУ. Не потом.

### Git
- Ветки: `block/[название]` (Tier 2+)
- Коммит после каждого логического блока работы
- Мёрж в main: только после валидации (микросимуляции ✅, стыки ✅)
- Мёрж по слоям снизу вверх

### Валидация
- Блочная: микросимуляции из контракта (вход → выход)
- Стыковая: формат + транспорт + версия совместимы
- Системная: GOAL.md критерии выполнены

### Эскалация
Кодер (2 попытки) → ревьюер-диагност → фоллбэк (мощная модель) → человек.
Эскалация — не провал, а нормальная часть процесса.

## Ритуалы

### Начало сессии
1. Прочитай GOAL.md
2. Прочитай BLOCKS.md — статус блоков
3. Прочитай последние записи changelog
4. Git status — есть ли uncommitted?
5. Определи задачу на сессию

### Завершение сессии
1. ФАКТ в changelog
2. Обновить BLOCKS.md (статусы)
3. Git: add, commit, push
4. Краткий итог: что сделано, что осталось, блокеры
